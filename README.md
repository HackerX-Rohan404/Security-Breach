Security Breach Simulation

A dynamic cybersecurity simulation that demonstrates a realistic device hacking sequence with immersive visual and audio effects.

Features

· Real-time terminal animations with system breach messages
· Progress bars showing system compromise stages
· Interactive hacking modules (Camera, Location, Contacts, Gallery, WhatsApp)
· Matrix-style background animation
· Emergency visual effects and sound alerts
· Countdown timer with final hacked message
· Responsive design for all devices

Technologies Used

· HTML5, CSS3, JavaScript
· Web Audio API for sound effects
· CSS animations and keyframes
· Google Fonts (Orbitron, JetBrains Mono, Share Tech Mono, Rajdhani)

Usage

Simply open index.html in a modern web browser. The simulation will automatically start with emergency alerts and progress through various hacking stages.

Educational Purpose

This project is created for cybersecurity awareness and educational demonstrations only.
